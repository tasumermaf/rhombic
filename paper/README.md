# Papers

Two papers. Both build from the `rhombic` library. Both are reproducible
from source.

## Paper 1: The Shape of the Cell

**Title:** The Shape of the Cell: Empirical Comparison of Cubic and FCC Lattice
Topologies Across Graph Theory, Spatial Operations, Signal Processing, and
Embedding Retrieval

**Target:** arXiv cs.DS (primary), cs.IR (secondary)

### Building

```bash
cd paper/
pdflatex rhombic.tex
bibtex rhombic
pdflatex rhombic.tex
pdflatex rhombic.tex
```

Or upload `rhombic.tex` and `rhombic.bib` to [Overleaf](https://overleaf.com).

## Paper 2: Pure Number Architecture

**Title:** Pure Number Architecture: How Structured Edge Weights Amplify the
FCC Lattice Advantage

**Target:** arXiv cs.DS (primary)

### Building

```bash
cd paper/
pdflatex rhombic-paper2.tex
bibtex rhombic-paper2
pdflatex rhombic-paper2.tex
pdflatex rhombic-paper2.tex
```

Or upload `rhombic-paper2.tex` and `rhombic-paper2.bib` to [Overleaf](https://overleaf.com).

## Figures

Both papers share the `figures/` directory. Paper 1 figures are generated by
`../scripts/generate_paper_figures.py`. Paper 2 figures are generated by
`figures/generate_paper2_figures.py`.

Figure PNGs are tracked in git (GitHub renders them, README references them).
Figure PDFs and compiled paper PDFs are gitignored — build from source.
